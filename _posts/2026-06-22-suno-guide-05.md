## Custom Modeでできること（一覧）

| 機能 | 概要 |
|------|------|
| **歌詞の手入力** | 自分で書いた歌詞を歌わせる |
| **スタイル指定** | ジャンル・楽器・雰囲気を細かく指定 |
| **Extend（延長）** | 既存クリップの末尾から続きを生成 |
| **Remaster** | 音質・ミックスのクオリティアップ |
| **Cover** | 既存楽曲を別スタイルで再生成 |
| **Inpaint（編集中）** | 指定した箇所だけ差し替え（β機能） |

## Extend（楽曲延長）の使い方

Sunoで生成される楽曲は通常1〜3分程度。Extendを使えばその続きを繋げて長い楽曲を作れる。

### 手順

1. 生成済みの楽曲を開く
2. 「**...**」メニュー → 「**Extend**」をクリック
3. 「Continue from this clip」が選択された状態で、Style・Lyricsを入力
4. 「Create」で末尾からの続きが生成される

### Extendのコツ

**コツ①：前のセクションタグを意識して継続させる**

Extendのlyricsに前のクリップの最後のタグを再指定することで、自然な流れが続く。

```
例：前のクリップが[Verse]で終わったなら
→ Extendのlyricsに[Chorus]から始めると自然につながる
```

**コツ②：InstrumentのExtendで長いアウトロを作る**

```
Lyrics欄に[Outro]とだけ入力し、Style欄に「fading out, instrumental」と追記
→ フェードアウト付きのアウトロが生成される
```

**コツ③：最大3〜4回のExtendが目安**

Extendを繰り返すと整合性が崩れていく。3〜4クリップ（合計5〜8分程度）を上限の目安にする。

## クリップの結合（Concatenate）

複数のExtendクリップを一つのMP3として書き出したい場合は、クリップを選択して結合する。

### 手順（2026年現在）

1. ライブラリから結合したいクリップを順に選択
2. 「**Create**」→「**Concatenate**」を選ぶ
3. 順番を並び替えて「Merge」
4. 合成されたクリップをダウンロード

> **注意：** Concatenateは再生成ではなく単純な結合のため、クリップとクリップの境目がノイズになることがある。境目のExtendで「自然につなぐ」ことを先に行うと品質が上がる。

## Remaster（音質向上）の使い方

生成された楽曲が音割れしていたり、バランスが悪いと感じたとき、Remasterで改善できる。

### 手順

1. 楽曲の「**...**」メニュー → 「**Remaster**」をクリック
2. 自動的に同じ楽曲のリマスター版が生成される（クレジット消費）
3. 元曲とリマスター版を比較して気に入った方を選ぶ

**Remasterが特に効果的なケース：**
- ボーカルが楽器に埋もれている
- 低音が強すぎる・高音が刺さる
- 全体的にモコモコして輪郭がぼやけている

---

## Cover（スタイル変換）の使い方

既存の楽曲（Sunoで生成した曲）を別ジャンルで作り直す機能。

### 手順

1. 楽曲の「**...**」メニュー → 「**Cover**」をクリック
2. 新しいスタイル（ジャンル・楽器編成等）を入力
3. 「Create」でスタイルを変えた新バージョンが生成される

**活用例：**
- ロック曲 → アコースティックバージョン
- J-POP → ジャズアレンジ
- ポップス → EDMリミックス

歌詞・メロディの大枠は維持しつつ、アレンジだけ変わる。

---

## Style欄の詳細な使い方

Custom ModeのStyle欄はSong Descriptionより**詳細な指定が可能**だ。

**有効なStyle欄の書き方（例）：**

```
city pop, female vocal, bittersweet, BPM 108, electric guitar, synth bass, 80s Japanese pop production, reverb-heavy drums, Casio tone synth pads
```

Song Descriptionとの違い：
- **Style欄**：楽器・プロダクション・テクニカルな指定に使う
- **Song Description**（Simple Mode）：ムードとジャンルの大まかな指示に使う

## まとめ

Custom Modeの応用機能をまとめると：

| やりたいこと | 使う機能 |
|-----------|---------|
| 曲を長くしたい | Extend |
| 複数クリップをまとめたい | Concatenate |
| 音質を改善したい | Remaster |
| 別アレンジを作りたい | Cover |
| 細かく楽器を指定したい | Style欄 |
