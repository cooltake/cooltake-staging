## ゴダイゴの成り立ち——バークリーで出会った天才たち

1977年、ゴダイゴが「モンキー・マジック」でデビューした時、日本の音楽シーンに衝撃が走った。英語と日本語を自在に行き来する歌詞、複雑なコード進行、プロフェッショナルなアンサンブル——これは当時の日本のポップスとは明らかに次元が違った。

### バークリー音楽大学での出会い

ゴダイゴの核となるのは**ミッキー吉野（キーボード）**と**スティーブ・フォックス（ベース）**の関係だ。

**ミッキー吉野**は1968年にアメリカのバークリー音楽大学へ留学。当時のバークリーは、ジャズ・フュージョンの本場として世界中から才能ある音楽家が集まる場所だった。彼はここでジャズ理論・アレンジ・作曲を徹底的に学び、日本のポップスとはまったく異なるハーモニー感覚と編曲力を身につけた。

**タケカワユキヒデ（ボーカル）**もバークリーで学んだ後、英語と日本語の両方で自在に歌える数少ない日本人アーティストとなった。

**ゴダイゴホーンズ**として知られるホーンアレンジは、岸本博氏（サックス）など一流ミュージシャンが参加することで、ゴダイゴのライブ・レコーディングに豊かなホーンセクションの彩りを加えた。こうした本格的なホーン編成は、当時の日本のポップバンドでは異例だった。

### ゴダイゴはフュージョンバンドか？

**ゴダイゴはフュージョンバンドではない**。これは重要な区別だ。

ゴダイゴのジャンルは**ポップ・ロック**であり、バークリーで培った音楽理論をポップスの文脈で使用した。「ガンダーラ」「モンキー・マジック」「Believe in Love」——これらはフュージョンではなく、洗練されたポップサウンドだ。

**フュージョン**（=ジャズ×ロック）の代表は：
- **カシオペア**（1976年結成）：ジャズ理論に基づく高速インストゥルメンタル
- **T-SQUARE**（1976年結成、旧名THE SQUARE）：メロディアスなフュージョン

これらは**1980年代**に全盛期を迎えたジャンルであり、1970年代後期のゴダイゴとは時代的にも音楽的にも異なる。

## カシオペア vs T-SQUARE——なぜこの2バンドが語られるのか

日本フュージョンを語る上でこの2バンドが必ずセットで語られる理由は、1980年代における「フュージョン界の2大巨頭」という立ち位置にある。両バンドはほぼ同時期にメジャーデビューし、競い合うようにファンを増やしていった。

| 項目 | カシオペア | T-SQUARE |
|------|----------|---------|
| サウンド | 硬質、タイト、テクニカル | メロディアス、ポップ |
| ギタリスト | **野呂一生**（刻みの速さと正確さが特徴） | 安藤まさひろ（メロディ重視） |
| 代表曲 | 「ASAYAKE」「GALACTIC FUNK」 | 「TRUTH」「ADVENTURES」 |
| 世界的評価 | テクニカルギタリストに高評価 | F1テーマで国際的知名度 |
| 活動スタイル | インスト中心、テクニック勝負 | 歌モノも取り入れた幅広さ |

**カシオペアの野呂一生**（のろかずなり）——しばしば「野呎孝司」と誤記されることがあるが、正しくは「野呂一生」だ。彼のギターは16分音符を正確かつ高速に刻むスタイルで、当時の日本のギタリストに多大な影響を与えた。

T-SQUAREの「TRUTH」（1987年）はフジテレビのF1中継テーマとして使われ、1987〜1991年の4年間、日本中に流れた。「TRUTH」は日本フュージョンの最大の輸出曲となり、海外でもフュージョンファンの間で広く知られている。

## 機材——フュージョン時代のサウンドを支えた機器

### キーボード革命

| 機材 | 特徴 | 現代の代替 |
|---|---|---|
| **Minimoog** | 太いリードシンセ音。フュージョンのソロに多用 | Arturia Mini V / Moog Software |
| **Rhodes Mark II** | ジャズ・フュージョンの代名詞エレピ | Arturia Piano V、Rhodes公式プラグイン |
| **Yamaha CP-70** | 生ピアノとエレピの中間。ゴダイゴで多用 | Pianoteq 8のモデリング |
| **ARP Odyssey** | リードとベースに使えるシンセ | Korg ARP Odyssey（復刻）、プラグイン版あり |
| **Roland Juno-60** | 1982年登場。フュージョン第2世代を支えた | Roland Cloud Juno-60 |

### ギタリストの機材

野呂一生（カシオペア）、安藤まさひろ（T-SQUARE）が多用したのは：
- **Ibanez カスタムギター**（日本製のクオリティがこの時代に急上昇）
- **Roland JC-120（ジャズコーラス）**：クリーントーンアンプの定番
- **Eventide Harmonizer**：ピッチシフターを使った空間系エフェクト

## フュージョンサウンドの特徴

<div style="background:rgba(255,255,255,0.10);border-radius:12px;padding:1.5rem;margin:1.5rem 0;border:1px solid rgba(56,189,248,.3);">
<h3 style="color:#bae6fd;margin-top:0;">🎹 フュージョンサウンドの核心</h3>
<ul style="color:#f1f5f9;">
<li><strong>複雑なコード進行</strong>：7th・9th・sus4コードを多用したジャジーなハーモニー</li>
<li><strong>16ビートのグルーヴ</strong>：ファンク由来のタイトなリズム感</li>
<li><strong>ユニゾンフレーズ</strong>：ギター+キーボードが同じフレーズを弾く</li>
<li><strong>クリーンなサウンド</strong>：歪みを使わない、抜けの良いトーン</li>
</ul>
</div>

フュージョンで多用されるコード進行の例：

```
Dm9 → G13 → Cmaj9 → Am7 → Dm9...（リズムチェンジ系）
Em7(b5) → A7(#9) → Dm9（モーダル・インターチェンジ）
```

## ゴダイゴ・フュージョンが切り開いたもの

この時代のミュージシャンたちが日本に持ち込んだものは、単なる「音楽スタイル」ではなかった。**音楽を論理的に理解し、技術として習得するという姿勢**——それが後のJ-POP、ゲーム音楽、アニメ音楽の高いクオリティを支える礎になった。
