## Sunoはスタート地点になれる

[第8回](/production/suno-guide-08/)まででSuno単体での制作技術を学び切りました。第9回からはChapter 4「外の世界へ持ち出す」です。

Sunoで生成した楽曲は、そのまま使うだけでなく**DAWに読み込んで仕上げる土台**としても使えます。この章では、SunoとDAWを組み合わせてより完成度の高い楽曲を作る方法を解説します。

---

## SunoのオーディオエクスポートとDAW読み込みの流れ

```
Sunoで生成 → MP3/WAVをダウンロード → DAWに読み込む → 編集・仕上げ → 完成
```

### ステップ1：Sunoからオーディオをダウンロード

1. 気に入った楽曲の「…」メニューを開く
2. 「Download」を選択
3. **形式の選択**:
   - **MP3**: 無料プランで利用可能。ファイルサイズが小さい
   - **WAV**: Pro/Premier プランで利用可能。ロスレスで編集向き

| プラン | 利用可能な形式 |
|-------|-------------|
| Free | MP3のみ |
| Pro | MP3・WAV |
| Premier | MP3・WAV・ステム分離 |

**DAWで本格的に編集するならWAVを強くおすすめします。** MP3は圧縮音源のため、DAWで処理を重ねると劣化が目立つことがあります。

---

## ステップ2：DAWに読み込む

### 対応しているDAW

SunoのオーディオはどのDAWにも読み込めます。

| DAW | 対応OS | 特徴 |
|-----|--------|------|
| GarageBand | Mac/iOS | 無料。初心者に最適 |
| Logic Pro | Mac | プロ仕様。¥36,800 |
| Ableton Live | Win/Mac | ループ制作・ライブ向き |
| Cubase | Win/Mac | 定番DAW。安定性が高い |
| FL Studio | Win/Mac | ビートメイク・EDM向き |
| Reaper | Win/Mac | 低価格・高機能。¥7,000〜 |

### 読み込み手順（GarageBandの例）

1. GarageBandを開き、新規プロジェクトを作成
2. Finderから**WAVファイルをドラッグ＆ドロップ**してトラックに配置
3. トラックが作成され、波形が表示される
4. 必要に応じてEQ・リバーブ・コンプレッサーを追加

---

## ステップ3：DAWで何ができるか

### できること

| 編集内容 | 説明 |
|---------|------|
| トリミング | 不要なイントロ・アウトロを切る |
| 音量調整 | セクションごとにボリュームを変える |
| EQ | 特定の周波数帯を強調・カット |
| リバーブ・ディレイ | 空間感を追加する |
| ボーカル追加 | 自分の声を録音して重ねる |
| 楽器演奏を追加 | MIDIや生演奏を重ねる |

### 重要な注意点

**SunoのオーディオはMIDIではなく音声データです。**

生成された楽曲の「コード進行」や「MIDI情報」をDAWで直接編集することはできません。Sunoのオーディオはあくまでも「録音済みの音声」として扱います。

---

## ステム分離の活用（Premier プラン）

Premier プランでは「ステム分離」機能が使えます。

### ステム分離とは

ステム分離とは、1つのミックスされた音源を**楽器別のトラック**に分離する技術です。

Sunoでは以下のステムに分離できます：
- **ボーカル**（Vocals）
- **インスト**（Instrumental）

### 活用例

**ボーカルだけ差し替えたい場合**:
1. ステム分離で「インスト」トラックをダウンロード
2. DAWに読み込む
3. 自分のボーカルを録音して重ねる

**インストBGMとして使いたい場合**:
1. ステム分離で「インスト」トラックをダウンロード
2. DAW不要でそのまま使える

---

## BGM制作のためのプロンプト例

BGM用途に特化したプロンプトをいくつか紹介します。

### YouTube動画用のBGM（穏やか・作業用）

```
lo-fi chill, soft piano, mellow drums, ambient, no vocals, instrumental,
relaxed, study music, 75bpm
```

### ゲームBGM（戦闘・アクション）

```
epic orchestral, driving drums, brass section, intense, cinematic,
action battle music, no vocals, instrumental
```

### Podcast・ナレーション用（背景に溶け込む）

```
ambient, light piano, subtle strings, no drums, quiet, background music,
corporate, professional, no vocals
```
