## 楽器制御の基本原則

Sunoに楽器を指定するには以下の3つのアプローチがある：

1. **Style欄でグローバルに指定**（全体のサウンドに影響）
2. **セクションタグと組み合わせてローカルに指定**（特定セクションの楽器）
3. **楽器タグを歌詞内に埋め込む**（位置指定）

---

## ギターソロの制御

### ギターソロを特定の位置に挿入する

```
[Verse]
（Aメロ歌詞）

[Chorus]
（サビ歌詞）

[Guitar Solo]

[Chorus]
（ラスサビ歌詞）
```

`[Guitar Solo]`タグをセクションとして配置すると、その位置に自然なギターソロが入る。

### ギターソロのスタイルを指定する

Style欄に追記：
```
shredding electric guitar solo, bluesy lead guitar, fingerpicking acoustic guitar solo
```

**効果的なギタースタイルタグ：**
| タグ | 音のイメージ |
|------|------------|
| `shredding guitar` | ヘビメタ系の速弾き |
| `bluesy lead guitar` | ブルースフィールのオーバードライブ |
| `clean Stratocaster` | クリーンなシングルコイル |
| `nylon string guitar` | クラシカルなガット弦 |
| `fingerpicking` | 指弾きアルペジオ |

## ドラムパターンの制御

ドラムのパターンは楽曲のグルーブの核心。以下のタグが有効：

### ドラムスタイルタグ

| タグ | 効果 |
|------|------|
| `four-on-the-floor kick` | EDM・ダンス系の4つ打ち |
| `heavy drum fills` | 派手なドラムフィル多め |
| `jazz brushes` | ジャズのブラシ奏法 |
| `breakbeat` | ブレイクビーツ（HipHop・ドラムンベース） |
| `minimalist drums` | シンプルなドラムパターン |
| `half-time feel` | スネアが2・4拍の裏に来るハーフタイム |
| `blast beat` | メタルコアの超高速ドラム |

### ドラムの音量バランス指定

```
punchy kick drum, tight snare, hi-hat-heavy groove
```

各ドラムパーツに形容詞をつけることで音の傾向が変わる：
- `punchy` / `boomy` / `tight` / `soft`
- `prominent` / `subtle`

## ベースラインの制御

ベースはグルーブと低音の核。以下のタグで操る：

### ベースタイプ指定

| タグ | 音のイメージ |
|------|------------|
| `walking bass line` | ジャズのウォーキングベース |
| `slap bass` | ファンクのスラップベース |
| `808 bass` | トラップ・HipHopの重低音 |
| `synth bass` | シンセサイザーベース |
| `fretless bass` | フレットレスベースの滑らかな音色 |
| `upright bass` | コントラバス（アコースティック低音） |

---

## ストリングス・ホーンのコントロール

### ストリングスの厚みを指定

```
lush string arrangement, sparse strings, pizzicato strings, soaring violins
```

| タグ | 効果 |
|------|------|
| `lush strings` | 厚いストリングスセクション |
| `sparse strings` | 薄いストリングス（目立ちすぎない） |
| `pizzicato strings` | ピチカート（弦をはじく奏法） |
| `string quartet` | カルテット（小編成） |

### ホーンセクション

```
brass section, jazz horn section, trumpet lead, saxophone solo
```

## セクション別楽器コントロール（上級）

セクションタグの中に楽器指示を埋め込む応用技：

```
[Verse - acoustic guitar and light drums]
（Aメロ歌詞：静かな始まり）

[Chorus - full band, heavy electric guitar]
（サビ歌詞：バンドサウンドで盛り上がる）

[Bridge - strings only, no drums]
（ブリッジ歌詞：静謐な雰囲気）

[Final Chorus - orchestra and choir]
（ラスサビ：壮大なフィナーレ）
```

このようにセクションタグの後ろに楽器指示を追記することで、セクションごとに異なる編成を誘導できる（精度は生成ごとにバラつく）。

## 実践：楽器コントロールまとめテンプレート

```
Style欄：
cinematic orchestral rock, BPM 132, electric guitar lead, driving drums with heavy fills, soaring strings, male vocalist, powerful and emotional

Lyrics欄：
[Intro - piano only]

[Verse - light drums, acoustic guitar]
（Aメロ）

[Pre-Chorus - add electric guitar, build up]
（Bメロ）

[Chorus - full orchestra, power chords]
（サビ）

[Guitar Solo]

[Chorus - choir added, maximum intensity]
（ラスサビ）

[Outro - fade out, strings only]

[End]
```
