## Sunoの歌詞入力2パターン

Sunoで歌詞を扱う方法は2通りある：

| 方法 | 特徴 |
|------|------|
| **自動生成**（Song Description任せ） | AI任せ。日本語率は低め。内容は毎回異なる |
| **Custom Mode で歌詞手入力** | 完全コントロール。日本語歌詞をそのまま使える |

日本語の歌詞を確実に使いたい場合は、**Custom Modeで歌詞を自分で書いて入力**するのが正確。

---

## Sunoが歌いやすい日本語の特徴

Sunoは日本語の音節（モーラ）を認識してメロディに乗せる。ただし以下の特性がある：

### 音節数が合わないとぎこちなくなる

英語の歌詞は音節を自然に伸縮させてメロディに乗せられるが、日本語は1音1モーラが原則。音節が多すぎると詰め込まれ、少なすぎると間延びする。

**目安：** 1フレーズあたり8〜14モーラ（文字）程度

```
✅ 良い例（11文字）：
夜が明けたら また会いに行く

❌ 詰め込みすぎ（17文字）：
夜が明けたらもう一度だけあなたに会いたい
```

### 伸ばし音・促音は活用する

「〜」や「っ」はリズムの調整に使える。

```
✅ 夜が明けた〜ら また会いに行く
✅ もう一度 もう一度 振り返って
```

## 韻の踏み方

Sunoは韻を踏んだ歌詞をメロディに乗せるのが得意。意識的に韻を踏むと完成度が上がる。

### 行末韻（末尾の音を揃える）

```
[Chorus]
君の笑顔が眩しくて（くて）
言葉が出なくなってしまって（くて）
このまま時間が止まればいいのに（のに）
また夢を見てた気がしてきたよ（よ）
```

行末の音をグループで揃えると、AIがメロディのフレーズを自然に繰り返す。

### ヘッドライン韻（行頭の音を揃える）

```
[Verse]
光を追いかけて
光の中で消えた
光は眩しすぎて
光になりたかった
```

行頭に同じ言葉を使う「アナフォラ」はポップスの定番テクニック。Sunoとの相性も良い。

## 日英ミックスの活用

J-POPの定番スタイルである日英ミックスは、Sunoでも有効だ。

```
[Chorus]
もう戻れない あの夏の日々
searching for you in the light
君の声が聞こえる気がして
running through the midnight
```

**ポイント：**
- サビ冒頭は日本語で感情を掴み、英語でグルーブを出す
- 英語部分はシンプルな単語を選ぶ（Sunoが正確に発音できるように）
- 日本語と英語の比率は 7:3〜6:4 程度が自然

## 感情タグの活用（歌い方のコントロール）

歌詞の中にタグを挿入することで歌い方（感情・強さ）を誘導できる：

| タグ | 効果 |
|------|------|
| `(whispering)` | 囁くように歌う |
| `(shouting)` | 叫ぶように歌う |
| `(sad)` | 悲しげに歌う |
| `(powerful)` | 力強く歌う |
| `(harmonies)` | コーラスハーモニーを加える |

**例：**
```
[Chorus]
(powerful) もう限界だと思っていたけど
(whispering) それでもここにいる
```

## 歌詞作成のNGパターン

### NG①：句読点を多用する

```
❌ 君の、笑顔が、眩しくて、言葉が、出ない。
```
句読点が多いと、AIがリズムを誤解してブツブツした歌い方になる。

### NG②：固有名詞を多用する

```
❌ 渋谷のスクランブル交差点で六本木ヒルズを背に
```
地名・固有名詞は発音が難しく、メロディに乗りにくい。

### NG③：難読漢字

```
❌ 黄昏の彷徨い
```
Sunoは難しい漢字の読みを誤ることがある。ひらがな・カタカナを積極的に使う。

```
✅ たそがれのさすらい
```

## 実践：歌詞テンプレート

以下のテンプレートをCustom Modeに貼り付け、歌詞部分を書き換えるだけで完成度の高い楽曲が生成できる：

```
[Intro]

[Verse]
（8〜12文字×4行。情景描写）

[Pre-Chorus]
（8〜12文字×2行。感情の高まり）

[Chorus]
（8〜14文字×4行。韻を踏む。繰り返しフレーズ入り）

[Verse]
（2番。1番と違う情景）

[Pre-Chorus]
（1番と同じかバリエーション）

[Chorus]
（1番と同じサビ歌詞）

[Outro]

[End]
```
