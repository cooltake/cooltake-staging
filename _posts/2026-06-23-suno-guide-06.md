## 「この楽器をもっと前に出したい」を実現する

[第5回](/production/suno-guide-05/)までで、Sunoの全パラメータを把握しました。ここからはChapter 3「楽器とフレーズを操る」です。

第6回のテーマは**楽器別コントロール**。「ギターをもっと前に出したい」「ドラムをなくしたい」「ストリングスを薄めにしたい」という要求を、Style欄のキーワードで実現する方法を解説します。

---

## 楽器コントロールの基本的な考え方

Sunoはキーワードの組み合わせでサウンドを推測します。楽器の存在感を変えるには以下の3種類のアプローチを使います。

| アプローチ | キーワードパターン | 効果 |
|-----------|----------------|------|
| **前面に出す** | `prominent [楽器]`・`[楽器] lead`・`heavy [楽器]` | その楽器をミックスの主役にする |
| **抑える** | `subtle [楽器]`・`minimal [楽器]`・`light [楽器]` | 存在は残しつつ控えめにする |
| **除外する** | `no [楽器]`・`without [楽器]` | その楽器を使わない |

---

## 楽器別キーワード集

### ギター

| 目的 | キーワード例 |
|------|------------|
| 前面に出す | `prominent electric guitar`, `guitar-driven`, `heavy guitar riff` |
| クリーントーン強調 | `clean electric guitar`, `jangly guitar` |
| 歪み強調 | `distorted guitar`, `crunchy guitar`, `overdriven` |
| アコースティック | `fingerpicked acoustic guitar`, `strumming acoustic` |
| 抑える | `subtle guitar`, `background guitar` |
| 除外 | `no guitar` |

**実例（ギター主役のロック）**:
```
indie rock, prominent electric guitar, guitar-driven, clean tone with slight overdrive,
bass, drums, male vocal, energetic
```

---

### ピアノ・キーボード

| 目的 | キーワード例 |
|------|------------|
| 前面に出す | `prominent piano`, `piano-led`, `piano ballad` |
| クラシカルな雰囲気 | `grand piano`, `classical piano`, `concert piano` |
| エレクトリックピアノ | `electric piano`, `Rhodes piano`, `Wurlitzer` |
| シンセ系 | `synth lead`, `analog synth`, `pad synth` |
| 抑える | `subtle piano`, `light piano accompaniment` |
| 除外 | `no piano`, `no keyboards` |

**実例（ピアノバラード）**:
```
piano ballad, prominent grand piano, piano-led, strings, female vocal, slow, emotional, melancholic
```

---

### ドラム・パーカッション

| 目的 | キーワード例 |
|------|------------|
| 前面に出す | `driving drums`, `heavy drums`, `prominent percussion` |
| タイトなビート | `tight drums`, `crisp snare`, `punchy kick` |
| ブラシドラム（ジャズ風） | `brush drums`, `jazz drums`, `soft percussion` |
| 電子ドラム | `drum machine`, `808 drums`, `trap drums` |
| 抑える | `minimal drums`, `subtle percussion`, `soft drums` |
| 除外 | `no drums`, `drumless`, `no percussion` |

**実例（ドラムなしのアコースティック）**:
```
folk, acoustic guitar, fingerpicked, no drums, no percussion, intimate, warm, gentle vocal
```

---

### ベース

| 目的 | キーワード例 |
|------|------------|
| 前面に出す | `prominent bass`, `bass-driven`, `funky bass` |
| スラップベース | `slap bass`, `funk bass` |
| シンセベース | `synth bass`, `808 bass`, `deep bass` |
| 抑える | `subtle bass`, `minimal bass` |
| 除外 | `no bass` |

---

### ストリングス・管楽器

| 楽器 | キーワード例 |
|------|------------|
| ストリングス（前面） | `prominent strings`, `lush strings`, `cinematic strings` |
| ストリングス（控えめ） | `subtle strings`, `string pad`, `light strings` |
| ブラス | `brass section`, `trumpet`, `trombone`, `no brass` |
| 木管 | `saxophone`, `flute`, `clarinet` |
| オーケストラ全体 | `full orchestra`, `orchestral arrangement` |

---

## 複数楽器のバランスを調整する

複数の楽器を組み合わせるときは、「主役1〜2楽器 ＋ 脇役楽器」という役割分担を意識します。

**例：ギターとピアノが共存するバンドサウンド**

バランスが悪い例（どちらが主役か不明）:
```
rock, electric guitar, piano, drums, bass
```

バランスを明確にした例（ギター主役、ピアノはサポート）:
```
rock, guitar-driven, prominent electric guitar, subtle piano accompaniment, driving drums, bass
```

## 実践：楽器コントロールを使ったプロンプト例

### 1. ジャズトリオ（ピアノ・ベース・ブラシドラム）

```
jazz trio, prominent piano, walking bass, brush drums, no guitar, no strings,
intimate jazz club atmosphere, swing feel
```

### 2. ストリングスカルテット（ドラムなし）

```
classical string quartet, prominent strings, no drums, no bass guitar, no synthesizer,
chamber music, elegant, emotional
```

### 3. ファンクバンド（スラップベース主役）

```
funk, slap bass driven, prominent bass, rhythm guitar, organ, tight drums,
no lead guitar solo, groovy, upbeat
```
