## SunoでSpotifyに配信できるか？

**結論：できる**。ただし条件がある。

| 条件 | 内容 |
|------|------|
| **Sunoのプラン** | **Proプラン以上**（¥1,200/月）が必要。無料プランの楽曲は商用利用不可 |
| **ディストリビューター** | RouteNote・TuneCore・DistroKid等の配信代行を使う |
| **著作権表示** | Sunoの規約に従い「Suno AI」のクレジット表示を推奨 |

| サービス | 費用 | 取り分 | 特徴 |

**RouteNoteを推奨**する理由：

2. AI生成楽曲の登録を明確に許可している
3. 配信先が多い（Spotify・Apple Music・Amazon・YouTube Music等）
4. Cooltake Studioがすでに登録済み（ID: 5318207）

## RouteNoteへの登録手順

### 1. アカウント登録

1. [routenote.com](https://routenote.com) にアクセス
2. 「Get Started Free」でアカウント作成（メールアドレスとパスワード）
3. メール認証を完了

### 2. アーティストプロフィール設定

1. ダッシュボード → 「Artists」→「Create Artist」
2. アーティスト名・プロフィール画像を設定
3. SpotifyのArtist IDと連携（既存Artistページがある場合）

### 3. 新しいリリースを作成

1. 「Releases」→「New Release」
2. シングル or アルバムを選択
3. 以下の情報を入力：

| 項目 | 内容 |
|------|------|
| Release Title | 楽曲タイトル |
| Artist | 作成したアーティスト名 |
| Release Date | リリース予定日（最低2週間後を推奨） |
| UPC/EAN | 自動生成でOK |
| Genre | 適切なジャンルを選択 |
| Language | 歌詞の言語 |
| AI Generated | **必ず「Yes」を選択** |

## 楽曲ファイルの要件

RouteNoteに提出するファイルの仕様：

| 項目 | 要件 |
|------|------|
| フォーマット | WAV（推奨）または MP3（320kbps以上） |
| サンプルレート | 44.1kHz または 48kHz |
| ビット深度 | 16bit または 24bit |
| 楽曲の長さ | 30秒以上 |
| ノーマライゼーション | -14 LUFS程度（ストリーミング標準） |

**Sunoからダウンロードする際：**
- Proプランで「WAV」ダウンロードを選択
- DAWでラウドネスを-14 LUFSに揃えてから提出

---

## アートワークの要件

| 項目 | 要件 |
|------|------|
| サイズ | 3000×3000px 以上（正方形） |
| フォーマット | JPEG または PNG |
| 内容 | アーティスト写真・イラスト・抽象デザインなど |
| 禁止 | URLや連絡先情報を含む画像 |

**アートワークをAIで作る場合：** Canva・Adobe Firefly・Stable Diffusionで作成可能。

## 収益の仕組みと目安

ストリーミングの1再生あたりの報酬（目安）：

| プラットフォーム | 1再生 | 1万再生 |
|----------------|------|---------|
| Spotify | $0.003〜$0.005 | $30〜$50 |
| Apple Music | $0.007〜$0.01 | $70〜$100 |
| Amazon Music | $0.004〜$0.007 | $40〜$70 |

**現実的な見通し：**
- ファンゼロの状態からの月収は$1〜数十程度
- SNS・YouTubeとの連動でリスナーを獲得することが先決
- AI楽曲でもバイラルヒットした事例は存在する（特にTikTok連動）

## 注意事項：著作権と規約

### Sunoの規約（2026年現在）

- **無料プランの楽曲**：商用利用・配信不可（個人使用のみ）
- **Proプラン以上の楽曲**：商用利用・配信可能
- **著作権帰属**：制作者（あなた）に帰属するが、SunoのAIモデルで生成したことの明記を推奨

### RouteNoteのAI楽曲ポリシー

- 「AI Generated」の項目で**必ずYesを選択**すること
- 虚偽申告は配信停止の原因になる
