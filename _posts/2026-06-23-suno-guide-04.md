## 歌詞を「書く」のか「AIに任せる」のか

[第3回](/production/suno-guide-03/)でセクションタグの使い方を学びました。第4回は、そのタグと歌詞を組み合わせてボーカル入りの楽曲を作る方法を解説します。

Sunoの歌詞には2つのアプローチがあります。

| アプローチ | 特徴 | 向いている場面 |
|-----------|------|--------------|
| **AIに全部任せる** | タグだけ書いてAIが歌詞を生成 | アイデア出し・英語曲・雰囲気重視 |
| **自分で書く** | 歌詞を自分で書いてSunoが歌う | 日本語曲・メッセージを伝えたい・世界観を固めたい |

どちらも使えますが、日本語の歌詞を意図通りに歌わせたい場合は「自分で書く」方が確実です。

---

## 歌詞の基本構成

ポップス・ロック系の楽曲では以下の構成が一般的です。

```
[intro]           ← 歌詞なし（楽器のみ）

[verse]           ← Aメロ：物語の導入・状況描写
（歌詞4〜8行）

[pre-chorus]      ← Bメロ：サビへの助走（省略可）
（歌詞2〜4行）

[chorus]          ← サビ：一番伝えたいメッセージ
（歌詞4〜8行）

[verse]           ← 2番Aメロ：物語の展開
（歌詞4〜8行）

[chorus]          ← サビ繰り返し

[bridge]          ← ブリッジ：転換・クライマックス前の山場
（歌詞2〜4行）

[chorus]          ← ラストサビ

[outro]           ← フェードアウト・締め
```

## 良い歌詞を書くポイント

### 1行の文字数を揃える

1行が長すぎると音節が合わず不自然に歌われます。英語なら1行6〜10単語、日本語なら1行10〜16文字が目安です。

❌ 長すぎる例:
```
今日も一日仕事が終わって家に帰ってくるとあなたの笑顔が待っていた
```

✅ 整えた例:
```
仕事終わりの帰り道
玄関に灯る明かり
```

### サビは回を追うごとに深める

サビは同じ文章構成を基本にしながら、**回ごとに単語を置き換えて情緒・対象を膨らませる**のが理想です。

- **1番サビ**: 感情・状況の最初の提示
- **2番サビ**: 同じ構成のまま、単語を差し替えて対象や情緒に広がりを持たせる
- **ラストサビ**: 曲全体の総まとめ。単語を置き換えてさらに深める手もあるが、**1番または2番のサビをそのまま繰り返すことで言いたいことを強め、パワーを持たせることも有効**

❌ 3回とも同じ歌詞をコピーするだけでは、聴き手の感情が動かない。

✅ 構成はそのままに、単語を入れ替えて重ね、ラストは繰り返しでパワーを出す:

```
[chorus] ← 1番：感情の提示
どこへ行っても君がいる
離れていても君がいる
この歌声が届くなら
もう一度だけ会いたい

[chorus] ← 2番：同じ構成・対象が「君」から「記憶」へ広がる
どこへ行っても消えない
忘れようとしても消えない
この痛みが証明する
まだ君を愛してる

[chorus] ← ラストサビ：1番を繰り返してパワーで締める
どこへ行っても君がいる
離れていても君がいる
この歌声が届くなら
もう一度だけ会いたい
```

### ブリッジで転換を入れる

ブリッジは曲の中間で雰囲気を変える場所です。短く、感情を凝縮させます。

```
[bridge]
もう戻れないとわかってる
それでも歩き続けるよ
```

## 日本語歌詞を使うときの注意点

### 崩れやすいパターンと対策

Sunoは英語ネイティブのモデルなので、日本語歌詞は英語に比べて不安定になることがあります。以下のパターンに注意してください。

**① 漢字が多すぎると読み飛ばされる**

❌ 問題が出やすい例:
```
深夜零時彷徨哀愁
```

✅ ひらがな・カタカナを混ぜる:
```
しんやゼロじ さまよう あいしゅう
```

**② 1行が長いと途中で音が乱れる**

1行を短くして、行数を増やす方が安定します。

**③ 英語の単語を混ぜると安定しやすい**

完全な日本語より、英単語を自然に混ぜたほうがSunoが歌いやすい場合があります。

```
[chorus]
My heartbeat 止まらない
君のことが ignite
```

## 実践：日本語ボーカル曲の例

### Style欄

```
j-pop, piano and strings, emotional ballad, female vocal
```

### Lyrics欄

```
[intro]

[verse]
春の終わりのホームで
最後の電車を待ってた
あなたが送ってくれた
短いメッセージひとつ

[pre-chorus]
言葉にできなかった
あの日の気持ちが

[chorus]
今でもここにあるよ
消えないまま輝いてる
忘れたいのに忘れられない
あなたとの時間

[verse]
夏が来ても秋になっても
同じ場所に立ってる
変わったのは季節だけで
心はまだあの日のまま

[pre-chorus]
もう会えないとしても

[chorus]
それでもここにあるよ
色褪せないまま輝いてる
手放したいのに手放せない
あなたとの記憶

[bridge]
前を向くと決めたのに
振り返るたびに気づく
この痛みさえ愛おしい

[chorus]
今でもここにあるよ
消えないまま輝いてる
忘れたいのに忘れられない
あなたとの時間

[outro]
```

### Title欄

```
あなたとの時間
```

## インストゥルメンタル（歌詞なし）にする方法

ボーカルなしのインストが欲しい場合は、Style欄に `instrumental` または `no vocals` を追加します。

```
lo-fi hip hop, piano, soft drums, instrumental, no vocals
```

セクションタグだけ書いて歌詞を空にしても同様の効果があります。
