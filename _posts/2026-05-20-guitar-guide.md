ギターを始めるとき、最初の選択は「エレキ」か「アコギ」かです。どちらも正解はありません。目指す音楽ジャンルと生活環境で決まります。

| | エレキギター | アコースティックギター |
|--|------------|---------------------|
| **必要機材** | ギター＋アンプ（最低限） | ギター本体のみ |
| **向いているジャンル** | ロック・メタル・ポップス・ジャズ | フォーク・弾き語り・J-POP |
| **初期費用** | ¥20,000〜（セット購入推奨） | ¥15,000〜 |
| **夜間練習** | アンプなしで可（ヘッドフォン利用も） | 生音が大きいため注意 |
| **指への負担** | 弦が細く押さえやすい | 弦が太く最初は指が痛い |

**弾きたい曲をイメージする** のが一番の判断基準です。バンド志向ならエレキ、一人で弾き語りたいならアコギを選びましょう。

## 必要な周辺機器

### エレキギターを選んだ場合

| 機材 | 用途 | 目安価格 |
|------|------|---------|
| ミニアンプ | 音を出す（自宅練習用） | ¥3,000〜8,000 |
| シールドケーブル | ギターとアンプを繋ぐ | ¥1,000〜3,000 |
| チューナー（クリップ型） | チューニング確認 | ¥1,000〜2,000 |
| ピック | 弦を弾く | ¥100〜500（数枚） |
| ストラップ | 立って弾く場合に必要 | ¥500〜2,000 |

※スターターセット（¥20,000〜25,000）なら上記が一括で揃います。バラバラに買うより安くなる場合も多いです。

### アコースティックギターを選んだ場合

| 機材 | 用途 | 目安価格 |
|------|------|---------|
| チューナー（クリップ型） | チューニング確認 | ¥1,000〜2,000 |
| ピック（厚さ違いを数枚） | 弦を弾く | ¥500 |
| カポタスト | キーを変えて弾く | ¥1,000〜3,000 |
| ストラップ | 立って弾く場合 | ¥500〜2,000 |
| ギタースタンド | 保管・展示用 | ¥1,000〜3,000 |

アコギはアンプ不要でシンプル。チューナーとピックだけあればすぐに始められます。
